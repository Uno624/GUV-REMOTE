static มีหลายความหมาย เช่น
       ถ้าประกาศในฟังก์ชัน loop(){static int x = 0;} = ประกาศครั้งเดียว ทำให้การวนรอบถัดไปถ้ารอบที่แล้วมีการเพิ่มค่า มันจะประกาศตัวเองเป็น 0 (static int x = 0;)
       static กับตัวแปร global (นอกฟังก์ชัน) ตัวแปรนี้ใช้ได้เฉพาะในไฟล์.cpp นี้
       static กับฟังก์ชัน static void ใช้เพื่อล็อกให้ฟังก์ชัน ไฟล์อื่นเรียกไม่ได้
